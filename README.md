2. 시작
- Village 당 게임 3개 => 총 6개
- Village 1 게임 3개 모두 Clear -> Village 2 게임 3개 모두 Clear -> End
- Village 당 생명 3개
- Village 1 Clear -> Village 2 중 생명 다 닳으면 ->  다시하기 씬(Village 1 end 지점부터 다시 시작)
- Village 1 게임 중 생명 다 닳으면 ->  다시하기 씬(처음부터 다시 시작)
